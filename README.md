# Sunnystep AI Agent Workshop

Workshop toolkit for the Sunnystep marketing team. March 4, 2026.

## What's in here

| Folder | Contents |
|--------|----------|
| `espanso/` | Keyboard macro definitions — interaction patterns for working with your agent |
| `ghostty/` | Terminal configuration — theme, font, keybindings |
| `prompts/` | The initial prompt template for agent-building sessions |

## Setup

Your facilitator will send you a single prompt to paste into Claude Code. That prompt reads this repo and configures everything automatically. You don't need to clone or download anything manually.

## Quick reference — macros

After setup, these shortcuts work anywhere:

| Type this | What it does |
|-----------|-------------|
| `:para` | Agent paraphrases back what it's about to do — confirms understanding before starting |
| `:interview` | Agent interviews you to figure out what to build |
| `:retro` | Agent reviews the session and produces concrete improvements for next time |
| `:explain` | Agent explains what just happened in plain language |
